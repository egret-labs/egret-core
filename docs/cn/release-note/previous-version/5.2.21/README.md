# 白鹭引擎 5.2.21 发布日志
白鹭引擎在 2019年6月11日，发布 5.2.21 稳定版本。


## 2D 渲染 - JavaScript 
- **[新增]**  `KTX` 纹理存储格式支持滤镜
- **[新增]**  `KTX` 纹理支持百度小游戏，小米快游戏，OPPO小游戏
- **[优化]**  优化微信小游戏上的内存管理
- **[修复]**  `URLLoader` 加载图片在微信小游戏上报错的问题

* 各小游戏平台对 `KTX` 纹理支持的状况:

系统/平台 | 微信小游戏 | 百度小游戏 | 小米快游戏 | OPPO 小游戏
------------- | ------------- | -------------| -------------| -------------
Android ETC1  | ✅ | ✅| ✅| ✅
iOS PVRTC  | ✅ | ❌ | ❌ | ❌
